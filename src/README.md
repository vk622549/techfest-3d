# 🌍 TechFest 3D Interactive Website

A modern 3D interactive website built using **Three.js**, **Vite**, **JavaScript**, **HTML**, and **CSS**. The project showcases a TechFest landing page with a rotating 3D Earth, smooth animations, and a responsive design.

---

## 🚀 Features

- 🌍 3D Rotating Earth using Three.js
- 🎨 Modern Hero Section
- 📖 About Section
- 🎯 Events Section
- 📞 Contact Section
- ✨ Smooth Scroll Animations (GSAP)
- 📱 Fully Responsive Design
- 💎 Glassmorphism UI
- 🖱️ Interactive Mouse Effects

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Three.js
- GSAP
- Vite

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/techfest-3d.git
```

Go to the project folder:

```bash
cd techfest-3d
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
